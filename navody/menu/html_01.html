1. Textový editor


V této kapitole se dozvíme jak pracovat s Visual Studio Code, ve kterém bude začínat programování naší webové stránky. Dozvíme se jak si tento editor nainstalovat a jak s ním pracovat.


Prvním krokem je si Visual Studio Code nainstalovat. Není to nic složitého a jedná se o klasické stažení aplikace. V našem 'menu' máte podstránku "Odkazy" ve které u prvního odkazu klikněte na dowload. Ihned (podle rychlosti vašeho internetu) vás tento odkaz přesměruje na oficiální stránky Visual Studio Code kde stačí kliknout na modré tlačítko 'download' (pokud máte jiný operační systém než Windows klikněte na šipečku vedle tlačítka 'download'). Soubor by se vám měl pak klasicky stáhnout do počítače. :)

Poté co bude ve vašem počítači Visual Studio Code připraven ke spuštění následujte další řádky. Editor si můžete spustit a na domovské stránce Visual Studio Code najděte bílý nápis Start a pod ním položku New File. Když na odkaz New File kliknete, zobrazí se vám nová stránka s řádkem 1 na které můžete začít pracovat. Poté co budete mít svojí práci dokončenou, stačí zmáčknout klávesovou zkratku Ctrl+S a soubor se vám uloží podle toho kam si ho vy chcete uložit. Doporučujeme si soubory ukládat do jedné složky. Doporučujeme také si soubor uložit již předtím než na něm začnete pracovat a ukládat s příponou .html
